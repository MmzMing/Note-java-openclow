# IDEA 快捷键

> 适用于 IntelliJ IDEA 2024+ 版本

## 插件推荐

| 类型 | 名称 | 作用 |
|------|------|------|
| 编写 | Key Promoter X | 快捷键教学助手：操作时提示快捷键，帮助记忆高频操作 |
| | String Manipulation | 字符串处理工具：支持大小写转换、加解密、排序等 |
| | GitToolBox | Git 增强：实时显示当前行最后提交信息 |
| | Rainbow Brackets | 彩虹括号：相同层级括号颜色一致 |
| | CodeGlance Pro | 代码缩略图：右侧显示代码全景图 |
| API查找 | RestfulTool | 树状展示项目中所有 RESTful API |
| Maven | Maven Helper | 可视化分析依赖冲突，一键排除 |
| 数据库 | MyBatisX | XML 与接口方法双向跳转 |
| AI | 阿里云-通义灵码 | 代码智能生成、智能问答 |
| 翻译 | Translation | 选中即可翻译 |
| JVM | jclasslib | 查看字节码 |
| | Profiler | 火焰图分析性能 |

## 编辑

| 快捷键 | 说明 |
|--------|------|
| `Ctrl + Space` | 基本代码补全 |
| `Ctrl + Shift + Space` | 智能代码补全 |
| `Ctrl + P` | 显示方法参数信息 |
| `Ctrl + Q` | 快速查看文档 |
| `Ctrl + /` | 行注释 `//` |
| `Ctrl + Shift + /` | 块注释 `/* */` |
| `Ctrl + W` | 递进式扩展选择 |
| `Ctrl + Shift + W` | 递进式收缩选择 |
| `Ctrl + D` | 复制当前行 |
| `Ctrl + Y` | 删除当前行 |
| `Ctrl + X` | 剪切 |
| `Ctrl + C` | 复制 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + Shift + ↑/↓` | 移动代码块 |
| `Ctrl + Alt + L` | 格式化代码 |
| `Alt + Enter` | 意图动作/快速修复 |
| `Ctrl + Alt + O` | 优化导入 |
| `Ctrl + Shift + Enter` | 补全语句 |
| `Ctrl + Alt + T` | 包围代码 |
| `Alt + J` | 多选相同单词 |
| `Alt + Shift + J` | 取消多选 |
