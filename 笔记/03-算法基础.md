## 递归

### 核心思想

自己调用自己，每次传入不同变量。
- 执行方法时创建新栈空间（独立内存）
- 局部变量独立，不相互影响
- 引用类型变量共享数据
- 必须向退出条件逼近，避免 StackOverflow

### 示例

```java
// 阶乘
public int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

// 斐波那契
public int fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

// 反转字符串
public String reverse(String s) {
    if (s.isEmpty()) return s;
    return reverse(s.substring(1)) + s.charAt(0);
}

```

## 排序

### 冒泡排序

时间复杂度 O(n²)，空间复杂度 O(1)

```java
public static void bubbleSort(int[] arr) {
    for (int i = arr.length - 1; i > 0; i--) {
        boolean swapped = false;
        for (int j = 0; j < i; j++) {
            if (arr[j] > arr[j + 1]) {
                int tmp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = tmp;
                swapped = true;
            }
        }
        if (!swapped) break; // 已有序，提前退出
    }
}

```

### 快速排序

时间复杂度 O(n log n)，空间复杂度 O(log n)

```java
public static void quickSort(int[] arr, int left, int right) {
    if (left >= right) return;
    int pivot = arr[left];
    int i = left, j = right;
    while (i < j) {
        while (i < j && arr[j] >= pivot) j--;
        arr[i] = arr[j];
        while (i < j && arr[i] <= pivot) i++;
        arr[j] = arr[i];
    }
    arr[i] = pivot;
    quickSort(arr, left, i - 1);
    quickSort(arr, i + 1, right);
}

```

### 归并排序

时间复杂度 O(n log n)，空间复杂度 O(n)

```java
public static void mergeSort(int[] arr, int left, int right) {
    if (left >= right) return;
    int mid = (left + right) / 2;
    mergeSort(arr, left, mid);
    mergeSort(arr, mid + 1, right);
    merge(arr, left, mid, right);
}

private static void merge(int[] arr, int left, int mid, int right) {
    int[] temp = new int[right - left + 1];
    int i = left, j = mid + 1, k = 0;
    while (i <= mid && j <= right) {
        temp[k++] = arr[i] <= arr[j] ? arr[i++] : arr[j++];
    }
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= right) temp[k++] = arr[j++];
    System.arraycopy(temp, 0, arr, left, temp.length);
}

```

## 数学技巧

### 判断连续整数

利用等差数列求和公式，无需遍历。

```java
/**
 * 判断版本号是否连续
 * 时间复杂度：O(n)，空间复杂度：O(n)
 * /
public static boolean isContinuous(List<Integer> versions) {
    if (versions == null || versions.size() <= 1) return true;

    Set<Integer> unique = new HashSet<>(versions);
    if (unique.size() != versions.size()) return false; // 有重复

    int min = Collections.min(unique);
    int max = Collections.max(unique);

    // 等差数列求和公式：S = (首项 + 末项) * 项数 / 2
    long expected = (long) (min + max) * unique.size() / 2;
    long actual = unique.stream().mapToLong(Integer::longValue).sum();

    return expected == actual;
}

// 示例
isContinuous(Arrays.asList(1, 2, 3, 4, 5)); // true
isContinuous(Arrays.asList(1, 2, 4, 5));    // false

```
